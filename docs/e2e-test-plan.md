## 交付物

1. **Playwright测试代码** (`e2e/`)
   - `page-objects/` - 页面对象
   - `tests/` - 测试用例
   - `fixtures/` - 测试数据
   - `playwright.config.ts` - 配置文件

2. **测试报告**
   - HTML测试报告
   - Allure报告（可选）

---

## 运行方式

```bash
# 安装依赖
npm install

# 运行所有测试
npx playwright test

# 运行指定模块测试
npx playwright test tests/auth

# 生成HTML报告
npx playwright show-report

# 使用Allure生成报告
npx allure serve
```
